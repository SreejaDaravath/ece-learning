- Practice 3-5 times

**2. During Presentation:**

**3. Handling Questions:**
- Listen completely before answering

- Link back to documentation
- Show willingness to learn

### **Common Questions to Prepare:**

**Technical:**
1. "Why did you choose vanilla JavaScript over frameworks?"

3. "How would you improve it?"

2. **Backup Plan** - Screenshots if demo fails

4. **Narrate Actions** - Explain what you're clicking
5. **Show Flow** - Complete journey from start to finish
6. **Highlight Features** - Point out unique aspects
